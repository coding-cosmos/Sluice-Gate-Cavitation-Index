# Sluice-Gate-Cavitation-Index


## File Structure
- data/ : Has all the .csv files for the dimensional as well as non dimensional data
- notebooks/ : Has all the google collab notebooks for different parts(SHAP, Sensitivity Analysis, Diagrams, Tables etc.)
- reference/ : contain all the reference material used in the project as base
- tables and figures/ : has .xlsx and .doc containing all the required figures and tables used throughout
- Paper.docx : final report
